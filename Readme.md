instanciable = publico
no instancaible = privado 

member function = funcion miembro, miembro de una clase

std::map: cuando necesitas buscar datos asociados a una clave y mantenerlos ordenados (fechas → precios).

std::stack: cuando solo trabajas con el último elemento insertado (LIFO), como en una calculadora RPN.

std::vector: cuando quieres acceso rápido por índice y recorrer muchos datos eficientemente.

std::deque: cuando necesitas un comportamiento parecido al de un vector, pero con inserciones eficientes también al principio.
